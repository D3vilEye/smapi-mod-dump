Peaceful Mining - 0.1.0

This mod makes the Stardew Valley mining experience much more peaceful.
It can either stop monsters from moving or stop them from appearing entirely.

Note that this mod does not provide a way to get monster drops if the monsters are disabled.

Requirements:
Stardew Valley Version 1.4
SMAPI Version 3.5.0

Defaults to stopping the monsters. Can be changed in the config.
The mod can also be disabled in the config.
