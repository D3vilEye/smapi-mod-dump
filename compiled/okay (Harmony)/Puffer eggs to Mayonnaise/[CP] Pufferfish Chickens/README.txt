This mod replaces all brown chickens with pufferfish chickens (a.k.a. Pufferchicks), all brown eggs with pufferfish eggs and adds a new pufferfish mayonnaise.

If you prefer the mod not to replace the icon in the Marnie's shop, go to the Config file and change the "DoReplaceMarnieShopIcon" parameter to "false". It is replaced by default.

Cheers!
